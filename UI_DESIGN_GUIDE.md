## 📏 Layout Specifications

### Typography
- **App Title**: 32px, Bold, Orange
- **Headers**: 18-24px, Bold
- **Body Text**: 14px, Regular
- **Small Text**: 12px, Light Grey
- **Button Text**: 16px, Bold, White
- **Link Text**: 14px, Colored (Orange for primary, Blue for secondary)

### Spacing
- **Page Padding**: 24px all sides
- **Section Gap**: 24px
- **Field Gap**: 16px
- **Component Padding**: 16px internal
- **Border Radius**: 12px (consistent across all inputs)

### Heights
- **Text Field**: 50px
- **Button**: 50px
- **Dropdown**: 50px  
- **Feature Card**: 120px (in grid)
- **AppBar**: 56px

### Colors
| Element | Hex Color | Usage |
|---------|-----------|-------|
| Primary Orange | #FF9800 | Buttons, icons, labels |
| Light Orange Fill | #FFE0B2 | Text field backgrounds |
| Success Green | #4CAF50 | Confirmation, uploaded state |
| Error Red | #F44336 | Error messages, alert |
| Dark Text | #212121 | Primary text |
| Light Grey | #9E9E9E | Hint text |
| White | #FFFFFF | Backgrounds, text on colored |

## ✨ User Flow

```
Start App
    ↓
┌─ [Choose]
├→ Email Login ─→ Email/Password ─→ Dashboard
├→ Phone Login ─→ Phone/OTP ─→ Dashboard  
└→ Sign Up ────→ Fill Form ─→ Select Society ─→ 
                  Select Flat ─→ Upload Docs ─→ 
                  Confirm ─→ Back to Login ─→
                  Login ─→ Dashboard
```

---

## 🎯 Key Design Principles

1. **Consistency**: All screens use same color scheme, fonts, spacing
2. **Clarity**: Icons + text for all actions
3. **Efficiency**: Multi-step form on one page (vertical scroll)
4. **Feedback**: Loading states, success messages, error handling
5. **Accessibility**: Good contrast, readable fonts, touch-friendly (50px min height)
6. **Brand**: Orange throughout for MyGate brand consistency
